cкрипт воссаздает каталог аннотаций (по указанному пути) из файла backup-a "annotations.json"

 используемые переменные:
JSON_SRC  = "data/back.orig/annotations.json"   # json file from backup 
DEST_PATH = "obj_train_data"                    # Каталог, в который нужно записать результирующие файлы
START_NUM = 700                                 # № первого фрейма батча

WIDTH  =2480     - ширина 
HEIGHT =3508     - высота изображеия в пикселах

Главные недостатки :
 1. Используется один размер изображения - для всех фрэймов. ПО факту , они могут быть разными
 2. Жесткий формат результирующих имен файлов (4.3: 0START_NUM.txt) - возможно, не пройдет для батчей с номерами  больше 1000 !?
